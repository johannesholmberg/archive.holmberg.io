About this website

This website was made for the purpose of creating a digital home to collect the writing and the work I do. The last couple of years I have downsized on my social media appearance and wanted to have a simple home where I can experiment at my own pace.

It’s built on Jekyll and hosted at MediaTemple. All CSS is written using Sass. The task manager I use is Gulp and Bower helps me keep track of packages. Icons come from the Streamline icon library. Photos are taken by Erik Carlsson. Questions? johannes@holmberg.io
